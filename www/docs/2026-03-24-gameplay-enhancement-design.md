# 小小环球旅行家 - 游戏玩法增强设计

> 创建日期：2026-03-24
> 目标用户：3-6岁幼儿 + 亲子互动

## 一、需求背景

### 核心痛点
- **识字困难**：3-6岁幼儿无法独立阅读文字内容
- **玩法单一**：目前只有颜色辨识游戏一种获取印章的方式
- **缺乏互动**：没有亲子合作/竞技模式

### 设计目标
1. 增加语音引导，解决识字问题
2. 增加多种小游戏，丰富玩法
3. 增加亲子互动模式（合作/竞技）
4. 增加创作功能，提升成就感

---

## 二、印章获取体系重构

### 印章类型与难度分级

| 印章类型 | 难度 | 游戏类型 | 适合年龄 |
|---------|------|---------|---------|
| 入境章🛬 | ⭐ 简单 | 点击类游戏 | 3-4岁 |
| 出境章🛫 | ⭐⭐ 中等 | 拼图/拖拽游戏 | 4-5岁 |
| 纪念章🌟 | ⭐⭐⭐ 挑战 | 翻牌配对/涂色创作 | 5-6岁 |

---

## 三、功能模块设计

### 3.1 语音引导系统

**功能描述：**
- 在关键位置添加喇叭图标 🔊
- 点击喇叭播报对应文字内容
- 不自动播放，完全由用户触发

**需要添加语音的位置：**

| 位置 | 播报内容 |
|-----|---------|
| 首页标题 | "小小环球旅行家，点击大洲开始探险吧！" |
| 大洲弹窗标题 | "这里是{大洲名}，有{数量}个国家等你探索" |
| 国家详情弹窗 | 国家名称、首都、货币、语言等基本信息 |
| 游戏规则说明 | 游戏开始前的规则介绍 |
| 获得印章提示 | "恭喜你获得了{国家名}的入境章！" |
| 勋章解锁提示 | "太棒了！你获得了{勋章名}勋章！" |

**技术方案：**
- 优先使用 Web Speech API（浏览器内置语音合成）
- 备选方案：使用预先录制的音频文件

**UI设计：**
- 喇叭图标放在文字旁边，尺寸适中（32px）
- 点击时有按下动画反馈
- 播放时喇叭图标有波纹动画

### 3.2 入境章游戏 - 点击国旗

**功能描述：**
- 最简单的游戏，适合3-4岁幼儿
- 国旗在屏幕上随机飘动，点击正确国旗即可

**游戏流程：**
1. 语音播报："找出{国家名}的国旗！"
2. 屏幕上显示4-6个飘动的国旗（包含目标国旗和干扰项）
3. 点击正确国旗 → 播放欢呼音效 → 获得入境章
4. 点击错误国旗 → 播放提示音 → 语音提示"再试试看"
5. 无限次机会，直到点对为止

**视觉设计：**
- 国旗像气泡一样轻轻飘动（CSS animation）
- 目标国家的正确国旗有微妙提示（边缘微发光）
- 点击成功时有烟花/星星特效
- 干扰项选择同大洲的其他国家国旗

**技术要点：**
- 复用现有的 `game-modal` 弹窗结构
- 复用现有的 `audioManager` 音效系统
- 国旗飘动使用 CSS `@keyframes float`
- 碰撞检测避免国旗重叠

### 3.3 出境章游戏 - 拼图

**功能描述：**
- 中等难度，适合4-5岁幼儿
- 将打乱的国旗碎片拼回原样

**游戏流程：**
1. 语音播报："把{国家名}的国旗拼好吧！"
2. 显示国旗的完整图案（参考图）
3. 下方显示打乱的碎片
4. 拖拽碎片到正确位置
5. 全部拼对 → 播放欢呼音效 → 获得出境章

**难度分级：**
- 简单：2×2 = 4块碎片（适合简单国旗如日本、法国）
- 中等：3×2 = 6块碎片（适合复杂国旗如美国、中国）

**视觉设计：**
- 碎片有磁性吸附效果（靠近正确位置自动对齐）
- 正确放置的碎片有绿色边框提示
- 完成时国旗整体闪光
- 参考图可以点击放大查看

**技术要点：**
- 使用 HTML5 Drag & Drop API 或 Touch Events
- 碎片使用 Canvas 或 SVG clip-path 生成
- 吸附效果使用距离阈值判断

### 3.4 纪念章游戏 - 翻牌配对

**功能描述：**
- 亲子竞技模式，两人轮流操作
- 记忆力挑战，策略性强
- 本地双人，同一设备面对面玩

**游戏流程：**
1. **游戏设置**
   - 选择参与的大洲（可选1-6个大洲）
   - 输入两位玩家名字（或选择旅行家）
2. **游戏界面**
   - 显示4×4翻牌区域（16张卡 = 8对国旗）
   - 顶部显示双方得分和当前回合
3. **翻牌规则**
   - 点击第一张牌 → 翻开显示国旗
   - 点击第二张牌 → 翻开显示国旗
   - 匹配成功 → 两张牌保持翻开，当前玩家得1分，继续翻牌
   - 匹配失败 → 两张牌翻回，换对方翻牌
4. **游戏结束**
   - 所有配对完成 → 显示最终得分和获胜者
   - 获胜者获得纪念章

**界面设计：**
```
┌─────────────────────────────┐
│  🏆 玩家1: 3分  vs  玩家2: 5分  │
│      [当前回合：玩家2]          │
├─────────────────────────────┤
│  🎴 🎴 🎴 🎴  │
│  🎴 🎴 🎴 🎴  │  ← 翻牌区域
│  🎴 🎴 🎴 🎴  │
│  🎴 🎴 🎴 🎴  │
└─────────────────────────────┘
```

**语音提示：**
- "轮到{玩家名}了！"
- "配对成功！"
- "配对失败，换{玩家名}"
- "游戏结束！{玩家名}获胜！"

**技术要点：**
- 国旗选择从已解锁的国家中随机抽取
- 翻牌动画使用 CSS 3D Transform
- 游戏状态使用独立的状态机管理

### 3.5 纪念章游戏 - 涂色创作

**功能描述：**
- 创意表达类游戏，培养艺术感
- 作品保存到画廊
- 可自由发挥，不强制正确配色

**游戏流程：**
1. 选择要涂色的国旗轮廓（从已解锁的国家中选择）
2. 显示国旗的线稿轮廓
3. 下方提供调色盘（包含该国旗的正确颜色 + 额外颜色）
4. 点击颜色 → 点击区域填充
5. 完成后点击"完成" → 保存到画廊 → 获得纪念章

**工具栏设计：**
```
┌─────────────────────────┐
│  🎨 涂色工具             │
│  [画笔] [橡皮] [撤销] [清空] │
├─────────────────────────┤
│  调色盘：                │
│  🔴 🟡 🔵 🟢 ⚫ ⚪ 🟣 🟠  │
├─────────────────────────┤
│      国旗线稿区域         │
│                         │
└─────────────────────────┘
│  [保存到画廊]  [重新开始]  │
└─────────────────────────┘
```

**保存内容：**
- 作品图片（Canvas导出为Base64）
- 对应国家ID
- 创作时间
- 是否为"正确配色"（可选成就）

**技术要点：**
- 使用 Canvas API 绘制
- 国旗线稿使用 SVG path 或 Canvas 绘制
- 填充算法使用 Flood Fill（洪水填充）
- 作品存储使用 localStorage

### 3.6 我的画廊页面

**功能描述：**
- 展示所有涂色作品
- 可回顾、删除、导出

**页面布局：**
```
┌─────────────────────────────┐
│  ←  我的画廊  🖼️            │
├─────────────────────────────┤
│  🎨 我的作品（5幅）          │
│                             │
│  ┌─────┐ ┌─────┐ ┌─────┐   │
│  │ 🇨🇳 │ │ 🇯🇵 │ │ 🇫🇷 │   │
│  │中国 │ │日本 │ │法国 │   │
│  │03/24│ │03/23│ │03/22│   │
│  └─────┘ └─────┘ └─────┘   │
│                             │
│  ┌─────┐ ┌─────┐           │
│  │ 🇺🇸 │ │ 🇧🇷 │           │
│  │美国 │ │巴西 │           │
│  │03/21│ │03/20│           │
│  └─────┘ └─────┘           │
│                             │
└─────────────────────────────┘
```

**交互功能：**
- 点击作品 → 放大查看详情
- 长按作品 → 弹出菜单（删除/导出）
- 导出功能 → 下载为PNG图片

**入口位置：**
- 首页底部增加"画廊"按钮
- 或护照本内的标签页

**数据存储：**
- 使用 localStorage 存储作品数据
- 导出时使用 Canvas toDataURL + download 属性
